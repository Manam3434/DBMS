# Employee SQL Database Project

A simple SQL project to manage and query employee and department records using MariaDB.



## Database Overview
- **DEPARTMENT Table**: Stores department names and IDs.
- **EMPLOYEE Table**: Stores staff details, jobs, salaries, and commissions.

## Main Features
- **Data Setup**: Scripts to create tables and insert sample employee data.
- **Table Maintenance**: Adding/deleting records and changing column sizes.
- **Salary Logic**: Automatically calculating 10% raises and annual salary projections.
- **Search & Filters**: Using pattern matching for names starting with 'S' or specific lengths.
- **Sorting**: Organizing reports by salary from highest to lowest.



## How to Use
1. Run the **Setup** queries to build the tables.
2. Run **Modification** queries to update salaries or records.
3. Run **Retrieval** queries to see the final reports.
